🗓️ Sistema de Agendamento de Mentoria
Este é um projeto de formulário estruturado para o agendamento de sessões de mentoria. O objetivo principal foi criar uma interface organizada, intuitiva e totalmente semântica, facilitando a coleta de dados pessoais, informações do mentor e disponibilidade de horários.

🚀 Tecnologias Aplicadas
Para este projeto, foquei na base sólida do desenvolvimento web moderno:

HTML5 Semântico: Uso rigoroso de tags como header, section, fieldset e legend para garantir acessibilidade e SEO.

CSS3 Customizado: Estilização externa via style.css (focado em layout e tipografia).

Google Fonts: Integração das fontes Poppins e Staatliches para um design moderno e profissional.

Validação Nativa: Implementação de atributos como required, minlength e tipos de input específicos (email, password, date) para garantir a integridade dos dados sem a necessidade inicial de scripts pesados.
